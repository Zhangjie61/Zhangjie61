大学生上课为什么一定要认真听讲？
你可以自己决定，问别人不如自己感受，现在的学校，是个开阔眼界的地方。聚精会神在这个时代已经是稀缺能力，大学应该打好基础，而走神是认真听讲的大敌。大学是让你通透地学一门专业的最好机会，在你掌握这些复杂的细节和规律并熟练应用之前，你没有资格谈大道至简的，因为深度和力度都不够。
